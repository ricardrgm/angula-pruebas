Ejemplos de Pipes

Convertir a mayúsculas:

Valor original:{{ cadena }}
Valor convertido: {{ cadena | uppercase }}

Convertir a minúsculas:

Valor original: {{ cadena }}
Valor convertido: {{ cadena | lowercase }}

Formatear un número:

Valor original: {{ numero }}
Valor convertido: {{ numero | number: "3.2-2" }}
Valor en euros: {{ numero | currency: "EUR":true }}
Valor en euros (ca-ES): {{ numero | number: "3.2-2":"ES" }}

Serializar un JSON:

Valor original: {{ personas }}
Valor convertido: {{ personas | json }}

Functions with Pipes

The hero's birthday is {{ birthday | date: getformat() }}

WELCOME TO ANGULAR:
date: {{ newdate | date: "full" }}
date: {{ newdate | date:"dd/MM/yy" }}

Capitalize un string (PIPE):

Valor convertido: {{ stringTocapitalize | capitalize }}

smart-truncate un string (PIPE):
Valor original: {{ stringToTruncate }}
